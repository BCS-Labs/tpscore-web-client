<h2>How to run project in dev mode</h2>

1. Install dependencies
```bash
yarn install
```
2. Add database credentials inside `.env.local`.
```
DB_HOST=db_host
DB_USER=db_admin
DB_PASSWORD=db_admin_password
DB_NAME=db_name
```
3. Run project
```bash
yarn dev
```
